Markdown Scala Reference

For the impatient: go to the spec.

This page represents the current efforts to convert the Scala Language Specification from LaTeX to Markdown, to ease future maintenance. While the majority of the work has been completed, there are a number of known issues and it is not yet ready to become the official version of the specification.

The default format for the specification is now HTML + MathJAX, single page, and should work in all modern browsers. The MathJAX rendering is slow, and we are looking to improve on this.

Alternative formats:

  • HTML + MathML, single page [Firefox/Safari, EXPERIMENTAL]
  • PDF [EXPERIMENTAL]
  • EPUB3 [EXPERIMENTAL]
  • Pandoc Markdown
